Group Six CSS Zen Garden Project

Group members: Cindi Nichols, Charleson Otubu and Virgil Cane

Our group's project theme will be outer space. The theme will be incorporated into the webpage through the color palette, fonts, and SVG images.

Two wireframes were created using wireframe.cc web application. The large wireframe was created to describe the layout of the webpage on a standard labtop monitor. The small wireframe was created to describe the layout of the webpage on a phone.
